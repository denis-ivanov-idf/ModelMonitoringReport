Initializes and returns a ClickHouse HTTP client based on the specified database and credentials file.

Parameters:

  • db (str) – The database to connect to. Supported options are “MM”, “PLAZO”, and “MX”. Defaults to “MM”.

  • db_cred_path (str) – The path to the credentials file. Defaults to “db_creds”.

  • **kwargs – Additional keyword arguments to be passed to the ClickHouse client.

Returns:

An instance of the ClickHouse HTTP client if the connection is successful, otherwise None.

Return type:

clickhouse_connect.driver.httpclient.HttpClient

Raises:

  • FileNotFoundError – If the credentials file is not found.

  • KeyError – If the specified database is not found in the credentials file.

Uploads a Pandas DataFrame to a ClickHouse database.

Parameters:

  • db_client – The ClickHouse client used to execute commands and insert data.

  • db_path (str) – The ClickHouse table path where the DataFrame will be uploaded.

  • pd_table (pd.DataFrame) – The Pandas DataFrame to be uploaded.

  • primary_key (str) – The primary key for the ClickHouse table. Defaults to “borrower_id”.

Returns:

Returns 0 if the upload is successful, or 1 if an error occurs.

Return type:

int

Raises:

  • ValueError – If the DataFrame contains unsupported data types.

  • RuntimeError – If there is an issue with creating the table or inserting data into ClickHouse.
